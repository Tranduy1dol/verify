# Generate os-latest.json
    `make gen-os`
# Run Madara and Pathfinder
    `make run-madara`
    `make run-pathfinder`
# Do transactions on Madara
    `make txs`
[!NOTE] Need to install `dojo`, `scarb`.
# Clear database
    `make clean`
### Before doing anything above, please run `git submodule update --init --recursive`.
